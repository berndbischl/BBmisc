stringsAsFactors {BBmisc}R Documentation

Convert all character columns in data.frame to factors.

Description

The function is DEPRECATED, use convertDataFrameCols!

Usage

  stringsAsFactors(data, do = TRUE)

Arguments

data

[data.frame]
The data.frame.

do

[logical(1)]
Should the conversion be done? Useful for omitting an if-statement in client code. Default is TRUE

Value

[data.frame].
